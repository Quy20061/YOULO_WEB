# 💜 YouLo – Kết nối thế hệ mới

> Ứng dụng nhắn tin & gọi điện realtime dành cho giới trẻ

## ✨ Tính năng

- 💬 **Nhắn tin realtime** — WebSocket, typing indicator, trạng thái đọc
- 📞 **Gọi thoại & video** — WebRTC hai chiều (cả 2 bên đều nghe được nhau)
- 🏠 **Bảng tin** — Đăng bài, like, bình luận
- 👥 **Bạn bè** — Gửi/chấp nhận lời mời kết bạn
- 🌙 **Giao diện Dark** — Thiết kế Gen Z, màu tím/hồng

## 🚀 Cài đặt & Chạy

```bash
# Clone repo
git clone https://github.com/NgoNgoc05-tech/DNU_FIT4007_CNPM_Nhom10.git
cd DNU_FIT4007_CNPM_Nhom10

# Cài dependencies
npm install
cd frontend && npm install && cd ..

# Chạy ứng dụng
npm run dev   # chạy cả server + frontend cùng lúc
```

## 🛠️ Tech Stack

- **Frontend:** React 18, Socket.IO-client, WebRTC API
- **Backend:** Node.js, Express, Socket.IO
- **Auth:** JWT + bcryptjs
- **Realtime call:** RTCPeerConnection (STUN servers)
